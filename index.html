PHP (Hypertext Preprocessor) adalah bahasa pemrograman sisi server yang digunakan secara luas untuk pengembangan web. PHP dapat terintegrasi dengan HTML untuk membuat halaman web dinamis. Materi dasar ini membentuk landasan untuk memahami cara kerja PHP dalam memproses informasi, berinteraksi dengan formulir web, mengakses database, dan menghasilkan konten dinamis pada halaman web
